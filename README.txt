[HƯỚNG DẪN CHẠY ỨNG DỤNG]

1. Cài Python nếu chưa có: https://www.python.org/downloads/

2. Mở terminal hoặc command prompt:
   - Tạo môi trường ảo:
       python -m venv venv
   - Kích hoạt:
       venv\Scripts\activate  (Windows)
       source venv/bin/activate (Linux/macOS)

3. Cài Flask:
       pip install Flask Flask-SQLAlchemy SQLAlchemy

4. Chạy ứng dụng:
       python app.py

5. Truy cập trình duyệt:
       http://127.0.0.1:5000
